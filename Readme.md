# Java String Operations

This repository contains Java programs that solve common string operation problems frequently asked in coding interviews and placement tests.

## Topics Covered

* Find the length of a string
* Reverse a string
* Check whether a string is a palindrome
* Count vowels and consonants
* Count words in a sentence
* Count the frequency of each character
* Remove spaces from a string
* Remove duplicate characters
* Find duplicate characters
* Find the first non-repeated character
* Check whether two strings are anagrams
* Replace vowels with a specific character
* Toggle the case of characters
* Count letters, digits, and special characters
* Find the longest word in a sentence
* Compress a string
* Reverse words in a sentence
* Check whether one string is a rotation of another
* Other interview-oriented string problems

## Technologies Used

* Java
* VS Code / IntelliJ IDEA
* Git & GitHub

## Purpose

This repository is created to practice Java string manipulation concepts, improve problem-solving skills, and prepare for coding interviews and placement tests.

## How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Compile and run any Java file.
4. Provide the required input (if applicable) and view the output.
